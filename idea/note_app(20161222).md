# note_app アイデア出し 12/22

* タスク管理ツール等との連携
  - 宿題、テスト範囲、教科書ページ、予定
* スマートフォンとの手書き連携
  - 図、ノートへの干渉

  - メモ、途中式、間違い

  - e.g. 色ペン、質感、定規、矢印、図形（強調）、行列（その他記号）、グラフ軸

  - e.g. 罫線、方眼、無地、縦線、黒板
* スマートフォンによる手書きの共有
  - e.g. 教え合い、採点
